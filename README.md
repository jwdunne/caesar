# Caesar

Caesar makes managing multi-repo projects easier.

## Install

### macOS

```bash
brew install caesar
```

### Windows

Download the MSI or...

```powershell
choco install caesar
```

or...

```powershell
scoop install caesar
```

### Linux (Debian)

```bash
apt install caesar
```

### Linux (RedHat)

```bash
yum install caesar
```
